遊び方

マウスを動かして移動します。
右端はゲージ類が存在しているため、移動できません。
左クリックで前方に弾を発射することができます。
右クリック長押しでバリアゲージを消費してバリアを展開することができます。
バリアを展開している間はダメージを受けません。
バリアゲージは時間経過で少しずつ回復します。
敵の攻撃や敵の機体に当たるとダメージを受けます。
５ダメージ受けてしまうとゲームオーバーです。
今の体力は体力ゲージで確認することができます。

敵は弾を飛ばして攻撃してきます。
ボス以外の敵は１撃で倒すことができます。
青い機体は普通の性能をしていますが、
黄色い機体は弾の射程が通常の３倍です。
赤い機体は弾の発射速度が２倍です。
ステージの最後にはボス機体が出現し、特別な弾を発射します。
体力が０になる前にボスを倒すことができれば、ゲームクリアです。
 
ゲームオーバー、ゲームクリア時もタイトルに戻って再度遊ぶことができます。
クリア時に表示されるスコアはボスも含めた敵機撃破数（最大５０）です。


注意事項

フォントはメイリオフォントを使用して日本語を表示しています。
タイトルは画像です。
音楽はありません。
画面サイズは一律で1500*1500になっているため、画面サイズがこれより小さい場合は正常に表示されません。
